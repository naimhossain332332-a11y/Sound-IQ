# 🔊 Sound IQ - Local AI Sound Searcher

**Sound IQ** is a free, lightweight, and 100% offline Audio Search Tool. It runs entirely on your local machine using Python and local AI, allowing you to search, index, and find any sound effect from your personal audio library simply by describing what you need.

---

## ✨ Features
* 🤖 **Local AI Search:** No API keys, no internet required. Describe the sound, and the AI finds it.
* 📁 **Instant Library Indexing:** Scan thousands of local audio files in seconds.
* 🔒 **100% Privacy-Focused:** Zero data tracking. Your audio files and search queries never leave your PC.
* ⚡ **Ultra-Fast Performance:** Optimized to run smoothly on mid-range hardware.

---

## 📥 Download & Installation

Choose your preferred way to install and run Sound IQ:

### 🔹 Method 1: One-Click Installer (Recommended)
1. Go to the [Releases](https://github.com/YOUR_USERNAME/YOUR_REPO_NAME/releases) section.
2. Download `Sound-IQ-Installer.exe`.
3. Double-click to run it (it will initialize in the background).
4. Close it, and check your **Desktop**—a fresh shortcut icon will be automatically created!

### 🔹 Method 2: Python Source Pack Setup (Open-Source)
1. Download `Sound-IQ-Source-Pack.zip` and extract it.
2. Open the extracted folder and double-click `setup.bat`.
3. The script will automatically install dependencies and create a Desktop shortcut for you.
4. You can also run the app directly anytime using `run.bat`.
